# NEPSE Stock Analysis (2010–2018)

Exploratory analysis of 16,029 Nepal Stock Exchange trading records 
across 11 companies — covering data cleaning, statistical analysis, 
and visualization.

## Tools Used
- Python, Pandas, NumPy, Matplotlib

## What's Covered
- Data cleaning (handling dirty entries, removing bad rows)
- Feature engineering (daily change %, price range, Up/Down/Flat labels)
- Company-wise performance comparison
- Volatility analysis using NumPy
- Visual dashboard with 6 charts
- Impact of 2015 Nepal Earthquake on NABIL stock

## Key Findings
- NLIC had the highest average closing price
- SCB had the most trading days recorded
- Approx 50% of all trading days across companies were 'Up' days
- 2015 Nepal Earthquake visibly impacted NABIL stock performance
